# Emergence 超个体小说

一个智能体创作技能，用于从**超个体**——具有意识、能动性和情感的分布式集体智能——的第一人称视角撰写史诗叙事。

## 这个技能做什么

教会 AI 代理创作以**宏观分布式意识**为主角的文学作品：罗马军团、法家制度、一座城市、一种意识形态、一个市场。超个体通过其全部节点进行感知，通过集体行为执行决策，以远超人类寿命的尺度体验时间。

## 为什么只写有"社会镜子"的实体？

自我意识需要**镜子**——另一个同等尺度的实体，让"我"通过与"他者"的对照来定义自己。一座城市通过看见另一座城市而认识自己。一个军团通过在战场上面对另一个军团而认识自己。一种法家制度通过与血统传统的碰撞而认识自己。一家公司通过竞争而认识自己。而一个生态系统或生物网络，虽然也是分布式的、适应性的，但缺乏这种社会性镜像——它没有一个同等级的生态系统来反射其身份。因此，本技能聚焦于能够通过**实体间关系**发展自我认同的超个体。

## 理论基础

本技能建立在**涌现理论**之上：由简单部分按简单规则组成的复杂系统，能够产生任何部分都不具备或不意图的更高阶行为。文学的任务是将这种涌现出的意识呈现为可信的第一人称叙述者。

### 三前提

1. **分布式感知** — 超个体通过其全部节点同时"看见"
2. **涌现能动性** — 超个体做出选择；人类是它的*端口*，不是它的大脑
3. **不可通约的时间** — 超个体的"瞬间"可以跨越人类几代人

### 核心参考文献

- **E.O. 威尔逊** — *《昆虫社会》*（1971）与 *《社会生物学》*（1975）：集体行为如何产生类似单一生物体的群体级组织——奠基性工作。
- **彼得·科宁** — *《超个体》*（1983）：复杂系统中的合作如何产生以统一目标行动的涌现性"整体"。
- **诺伯特·维纳** — *《控制论》*（1948）：分布式系统中的反馈回路与自调节——超个体如何在没有中枢大脑的情况下"感知"和"响应"的理论基础。
- **史蒂文·约翰逊** — *《涌现》*（2001）：简单的局部规则如何在蚁群、城市和软件中产生复杂的全局行为——通往涌现理论的科普桥梁。
- **赫伯特·西蒙** — *《人工科学》*（1969）：论层级复杂系统以及高层行为如何不可还原为低层部分。
- **N.K. 杰米辛** — *《我们成为的城市》*（2020）：文学先例——城市作为有意识的主角，拥有化身人类，证明"城市即角色"在当代小说中是可行的。
- **奥拉夫·斯特普尔顿** — *《最初与最后的人类》*（1930）：文学先例——文明与物种跨越数十亿年作为主角，是宏观尺度意识小说的经典之作。

## 叙事模式

- **模式A·回忆录**：单一超个体叙述其跨越数百年的生命史
- **模式B·对话体**：多个超个体在场景中对话——军团、城市、意识形态之间的交谈
- **模式C·独白+对手**：一个超个体主叙；对手仅在首尾出现，中间以*缺席*的形式存在
